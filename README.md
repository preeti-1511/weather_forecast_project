# Weather Forecast Project
# Description

```
A Ruby on Rails weather forecast application with geocoding, caching, and RESTful API. Features 30-minute cache by zip code, comprehensive error handling, and full test coverage.
```

```
Weather Forecast Application - A Ruby on Rails application that retrieves and displays weather forecast data for any address. Features include geocoding integration, 30-minute caching by zip code, RESTful API endpoints, comprehensive error handling, and full test coverage. Built with service-oriented architecture following best practices.
```

```
A complete Ruby on Rails 7.1 weather forecast application that accepts any address as input and retrieves current weather data including temperature, humidity, wind speed, and weather conditions. The application implements intelligent caching (30-minute TTL by zip code), provides both web interface and RESTful API endpoints, includes comprehensive error handling, and follows service-oriented architecture patterns. Features full test coverage with RSpec, Postman collection for API testing, and extensive documentation.
```

## Key Features to Highlight
- 🌍 Address-based weather forecast retrieval
- ⚡ 30-minute intelligent caching by zip code
- 🔄 RESTful API with JSON responses
- 🧪 Comprehensive test coverage (RSpec)
- 📚 Extensive documentation (API docs, architecture, setup guides)
- 🏗️ Service-oriented architecture with design patterns
- 🎨 Clean, modern web interface
- ⚠️ Robust error handling

## Tech Stack
- Ruby 3.1.4
- Rails 7.1.3
- SQLite3 (development)
- Geocoder gem for address geocoding
- HTTParty for API calls
- RSpec for testing
